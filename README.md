## General info
This project is a simple Campaign managament app for Dungeons and Dragons for Game-Masters and Players.
	
## Technologies
Project is created with:
* Ruby 2.6.1
* Rails 6.1.1
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
